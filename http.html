How Connecting to a web page works

  1. First, you enter the URL in the browser which points to a webpage (or clicking a link to a web page).
  2. The server is waiting for a request from the browser and is hosting all of the webpages.
  3. The browser creates a TCP/IP connection with the machine.
  4. The browser sends a request for just the domain name to the server.
  5. Once the domain is established, the browser then sends the request for the entire URL to the server.
  6. The server sends the web pages display back to the browser.
  7. In turn , the browser sends the web page display back to you rendered.

(Parlante, 2016)
